# 👥 Customer Segmentation Analysis

---

## 📌 Project Overview

This project analyzes customer data to group similar customers based on behavior and demographics. The goal is to enable targeted marketing, improve product recommendations, and increase customer engagement.

---

## 🛠️ Tools Used

* 🐍 Python
* 📊 Pandas
* 📈 Matplotlib / Seaborn
* 📉 Excel
* 💾 CSV
* 📊 Scikit-learn (Clustering algorithms)

---

## 📂 Dataset

* Customer dataset containing demographics, purchase history, and engagement metrics
* Features include:

  * `customer_id` 👤
  * `age` 👶👵
  * `gender` 🚻
  * `annual_income` 💰
  * `spending_score` 💳
  * `purchase_history` 🛒
  * `engagement_level` 📈

---

## 🎯 Project Objective

* Identify distinct customer segments
* Understand purchasing behavior and engagement patterns
* Support personalized marketing strategies
* Improve customer retention and satisfaction

---

## 🧹 Data Cleaning Steps

* Removed duplicate entries
* Handled missing values ❓
* Encoded categorical variables 🔢
* Normalized numerical features 📏
* Prepared data for clustering analysis 🗂️

---

## 📊 Analysis Performed

* Customer distribution by demographics 👥
* Visualized spending vs income patterns 💳💰
* Applied clustering algorithms (K-Means, Hierarchical) 🌲
* Determined optimal number of clusters using Elbow Method 🔍

---

## 📈 Key Insights

* Identified X distinct customer segments 🏷️
* High-spending customers cluster together 💰
* Certain segments are more responsive to marketing campaigns 📢
* Low engagement group identified for retention campaigns ⚠️
